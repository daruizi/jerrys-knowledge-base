## 第 0 步：必须要懂的准备工作（必看！）

在正式请 AI 过来帮忙前，我们需要了解几个基础概念。

### 1. 它是免费的吗？怎么收费？
**不免费**。如果是官方原版，你需要绑定海外银行卡的官方 Claude Pro 或 API 账号，门槛比较高。

**但别急！** 请直接跳到文档的 **[中国用户特供版](#-中国用户特供给-claude-做换脑手术省钱秘籍)** 章节，我们会保姆级地教你怎么连上国内便宜又强大的 AI 替身（比如 DeepSeek 或 智谱 GLM），不用翻墙，几毛钱就能用一整天！

### 2. 核心大前置：安装 Node.js (代码运行引擎)
Claude Code 这个软件需要安装在一个叫做 Node.js 的“底盘”上才能运行。如果你是一张白纸的新手，电脑里肯定没有它。
> **🛠️ 安装方法 (无脑狂点下一步即可)**：
> 1. 去 [Node.js 中文官网](https://nodejs.org/zh-cn) 下载左边的“长期维护版”(LTS)。
> 2. 下载下来就跟安装 QQ 一样，一路狂点 `Next(下一步)`，直到完成。不用配置任何东西！

### 3. “终端（Terminal）”是什么玩意儿？
你可以把终端想象成电脑的“底层指挥部”。平时咱们用鼠标点来点去，而在“终端”里，我们是用**输神秘代码**的方式让电脑干活。
> **💡 小白理解**：就是电脑里那个黑乎乎（或蓝乎乎）的、只能打字进去的神秘框框。我们和 Claude Code 的所有聊天都会在这里进行。

**如何召唤终端？**
- 🍎 **如果你是苹果电脑 (Mac)：**
  按 `Command ⌘ + 空格键` 唤出搜索，输入 `终端` 或 `Terminal`，回车打开。
- 🪟 **如果你是 Windows 电脑：**
  屏幕左下角“开始”菜单上点**右键**，选择 `Windows PowerShell (管理员)` 或者 `终端 (管理员)`。记得一定要带“管理员”三个字哦！屏幕弹窗问你是否允许，点“是”。

## 保姆级安装指引

### 第 1 步：装好工具
打开你刚才召唤出来的黑框框。

**⚠️ 【关键防坑】给下载器换上“国内淘宝高铁引擎”**
因为官方服务器在国外，直接安装大概率会卡在一半报错“网络连接超时 (ETIMEDOUT)”。我们先念一句“切换淘宝源”的咒语提速。
把这行字复制，在黑框框里单击鼠标右键粘贴，敲回车：
```bash
npm config set registry https://registry.npmmirror.com
```

**正式安装 Claude Code：**
继续把下面这行代码**复制粘贴**进黑框，敲回车：

```bash
npm install -g @anthropic-ai/claude-code
```
*(如果屏幕上开始刷刷刷地有彩色进度条闪过，最后安静下来没报错，恭喜你，安装成功啦！)*

### 第 2 步：进入你的代码文件夹（学会使用 cd 魔法）
安装好之后，你怎么告诉 AI 它应该看哪里的代码呢？你需要“走”进那个专门放代码的文件夹。
在黑框框里，我们用 `cd` 这个命令来走路。（cd 就是 Change Directory，改变目录的意思）。

**✨ 小白绝招：**
在黑框框里输入 `cd `（注意字母 d 后面**一定要敲一个空格！**），然后把你电脑里想要放代码的**文件夹，用鼠标直接拖拽到黑框框里**，按下回车！你看，你就进去了！

## 🎉 快速上手：你的第一次 AI 编程

当你成功用 `cd` 魔法走到你的项目文件夹后，就可以召唤黑框里的学霸小助手了！

直接在黑框框里输入这串咒语：
```bash
claude
```
按回车！

这时候会出现一个带有 `>` 的提示符，你就可以像用微信聊天一样打字了，比如：
> 🙋‍♂️ `我是一个完全不懂代码的新手，我现在想做一个算账的小工具网页，第一步我该怎么办？带我一步一步做。`

接下来你只需要看它表演。如果它说“我要创建/修改这个文件，你同意吗？（Y/n）”，你只需要按电脑键盘上的 `Y` (代表 Yes) 键，敲回车同意即可，它自己就把文件写好了！

## 🇨🇳 中国用户特供：给 Claude 做“换脑手术”（省钱秘籍）

官方原版要外币信用卡？网速很慢？没关系！Claude Code 是允许高度自定义的，我们可以把它换成国内的大模型（如 DeepSeek，智谱 GLM 等）。

> **🛠️ 原理：** 这个过程就像给助手做“换脑手术”——壳子还是好用的 Claude CLI，但内核连上了咱们国内申请好的“平价无障碍强力大脑”。

### 手术步骤：

**1. 找到“脑神经配置文件”**
用电脑自带的记事本，打开系统里的一个隐藏文件。这个文件在你电脑的用户目录下的 `.claude` 文件夹里，名字叫 `settings.json`。
> - **Mac 路径定位**：打开终端，输入 `open ~/.claude` 就能打开那个隐藏文件夹。 
> - **Windows 路径定位**：在任意文件夹上面的地址栏里直接粘贴输入 `C:\Users\你的用户名\.claude`（记得把“你的用户名”换成你电脑真实的用户名），敲回车。
> 
> *如果进去发现没有 `settings.json` 文件，就自己右键新建一个文本文档，改名叫这串英文就行。*

**2. 注入新大脑的配方**
用“记事本”打开 `settings.json` 文件，把下面的内容原封不动粘贴进去：

```json
{
  "apiProviders": [
    {
      "name": "deepseek-proxy",  
      "type": "openai",     
      "apiKey": "填入你去 DeepSeek 官网申请的 API Key",
      "baseUrl": "https://api.deepseek.com/beta",
      "defaultModel": "deepseek-coder"
    }
  ],
  "defaultProvider": "deepseek-proxy"
}
```
*(注意：请去 DeepSeek 或者其他国产模型官网免费注册并获取一串叫做 `API Key` 的密钥，填在上面的引号里哦。)*

🛑 **【极其致命的防坑警告】：余额与实名制！**
现在几乎所有国内大模型厂商，对于刚注册的“白嫖新账号”，API 接口都是**默认封禁死锁状态的**！
如果你拿到 Key 之后直接配上去用，终端里大概率会一直爆红字报错（如 `402 Payment Required` 或 `403`）。
**破解之法：** 拿到 Key 之后，必须在这个大模型厂商的后台，**完成实名认证**，并**充值人民币（哪怕只充 1 块钱 或 5 块钱即可）**，你的 API 接口才会瞬间打通复活！

**3. 保存退出并测试**
保存后关闭记事本。重新在黑框框里输入 `claude` 回车。
恭喜你！现在它已经用上国内便宜又好用的大脑啦！速度快到飞起飞！

## 核心功能详解：让它直接帮你干活

和传统的“你问我答”网页 AI 不同，Claude Code 是你的专属程序员，它能**直接操作你的电脑文件**！

### 1. 充当代码修理工（排错修 Bug）
如果网页打不开，或者有个功能坏了：
你完全不需要去理解那些红红绿绿的错误代码。直接把报错信息用鼠标复制粘贴给它，或者对它说：
> `刚才运行出错了，我不懂这个报错是啥意思。请你帮我查一下哪里写错了，直接修好它！`
它会自动去读相关文件，找出错误原因并帮你改好。

### 2. Git 游戏存档员（代码存档与合并）
写代码就像打游戏闯关，我们需要随时“存档”。万一写错了，还能读档重来。在程序员的世界里，存档叫做 `Git 提交 (commit)`。
你完全不需要记复杂的 Git 存档命令，直接对它说：
> `我刚刚把登录页面的背景做完了，帮我把刚才写的代码存个档吧，存档名字叫‘完成登录页背景’。`

### 3. 一句话大改造
当你想要改一些东西时，大胆下令：
> `帮我把这个项目里所有的按钮，都变成粉色圆角的可爱风格！`

## 🧠 CLAUDE.md：教 AI 懂你的项目（附极简模板）

AI 虽然聪明，但它不认识你，不知道你的喜好。
我们可以在代码文件夹里，自己新建一个名叫 `CLAUDE.md` 的记事本文件。这就是给 AI 看的“项目管家用工说明书”，它每次启动干活前，都会先默默读一遍这个文件。

### 📝 小白极简版 CLAUDE.md 模板
直接新建一个记事本，命名为 `CLAUDE.md`，然后把这段话粘贴进去：

```markdown
# 给 AI 助手的心理辅导

嗨！在这个项目里当你帮我写代码时，请严格遵守以下规则：
1. **必须说人话**：我是个完全不懂技术的新手小白。每次跟我解释代码原理、或者接下来要做什么时，必须要用生活中常见的例子（比如去餐厅点菜、盖房子等）来打比方。
2. **写全中文注释**：你帮我写的每一段代码，必须在旁边写上极其详尽的中文注释，说明这一句是干嘛的。
3. **不要只说不做**：发现问题请直接动手替我修改文件，不要只丢给我一堆思路或片段让我自己拼图。帮我把代码全部填好、改好放到文件里！
```

## 🆘 新手急救箱（新手排错指南）

新手上路遇到黑框框容易慌？深呼吸，这里有万能急救贴士。

### 💊 症状 1：黑框框提示“claude: command not found” （或是找不到命令之类的红字）
- **🩺 原因：** 电脑反应慢半拍，或者是找不到你刚才装的东西。
- **🚑 解决思路：** 把这个黑框框右上角点 X 关掉，重新打开一个新的，输入 `claude` 再试一次。如果还不行，**重启一下电脑能解决 99% 的安装问题！**

### 💊 症状 2：坏了！AI 把我的代码改废了，运行不起来了！怎么撤销？
- **🩺 原因：** 小助手有时候也会写错代码，越帮越忙。
- **🚑 解决思路：** 如果你刚才按照教程让它帮你“存档”过（使用过 Git），你就可以大喊一声：
  > `停止你的修改！帮我把所有代码撤销、恢复到我们最后一次存档的样子！刚才写的全作废！`

### 💊 症状 3：聊着聊着，AI 忽然好像得了“失忆症”，前言不搭后语？
- **🩺 原因：** 你们的聊天记录太长了，超过了 AI 脑子的“短期缓存记忆容量”。
- **🚑 解决思路：** 在聊天框里发送一条神奇斜杠命令： `/compact` （按回车）。这就相当于拍拍它的脑袋说“把前面的废话清理掉，记住核心就行”。它就会重新清醒。
- 如果彻底没救了，想重启一个新的大脑和它对话，请发送 `/clear` 清空记忆。

### 💊 症状 4：控制不了它，它一直在刷屏修改，怎么让它立刻停下来！
- **🩺 原因：** 你可能觉得在这个黑框里它没给你“暂停按钮”。
- **🚑 解决思路（拔电源法）：** 在黑框框里，不管三七二十一，**同时按下键盘上的 `Ctrl` 和字母 `C`**。这在程序员界叫“物理打断强制终止”，它会立刻闭嘴停止一切工作。
