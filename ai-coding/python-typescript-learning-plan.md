## Python 学习计划（2周）

### Day 1-3：语法核心（直接上手写）

```
必学：
├── 变量、数据类型
├── 条件判断 if/elif/else
├── 循环 for/while
```

```
└── 常用库：requests, json, pathlib
```

## TypeScript 学习计划（2周）

### Day 1-3：JS基础（TS的前置）

```
必学：
├── let/const vs var
├── 箭头函数 () =>
├── 解构赋值
├── 模板字符串 `${}`
├── Promise + async/await
└── 数组方法 map/filter/reduce（最重要）
```

### Day 4-7：TS核心

```
├── 类型注解 :string :number
├── 接口 interface
├── 类型别名 type
├── 泛型基础 <T>
└── 联合类型 | 交叉类型 &
```

### Day 8-14：实战项目

- 用Claude Code辅助写一个**CLI工具**或**API服务**
- 推荐：Hono/Fastify（轻量框架）
