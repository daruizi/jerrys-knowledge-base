## 第一阶段：Python 基础 + 数据抓取（Day 1-7）

### Day 1 — Python 环境 & 核心语法
- **概念（30min）**：安装 Python 3.12+（`brew install python`），了解 `venv` 虚拟环境，学习变量、类型、f-string、列表/字典推导式
- **实操（30min）**：创建项目目录，建虚拟环境，写一个脚本处理 JSON 文件（读取、过滤、输出）
- **交付物**：`day01/json_processor.py`

### Day 2 — 函数、模块、错误处理
- **概念（30min）**：函数定义与类型标注（type hints）、`*args/**kwargs`、模块/包导入机制、`try/except` 异常处理
- **实操（30min）**：写一个 CLI 工具，接收命令行参数，读取 CSV 文件并统计数据
- **交付物**：`day02/csv_analyzer.py`

### Day 3 — 面向对象 & 数据类
- **概念（30min）**：class、`__init__`、继承、`@dataclass`、`@property`、协议（Protocol）
- **实操（30min）**：用 dataclass 建模一个简单的任务管理器（增删改查 tasks）
- **交付物**：`day03/task_manager.py`

### Day 4 — 包管理 & HTTP 基础
- **概念（30min）**：`pip`、`requirements.txt`；HTTP 方法、状态码、请求头、Cookie；`httpx` 库
- **实操（30min）**：用 `httpx` 调用 GitHub API 获取数据，模拟带请求头的访问
- **交付物**：`day04/github_fetcher.py`

### Day 5 — 网页抓取基础（静态页面）⭐
- **概念（30min）**：HTML 结构与 CSS 选择器；`BeautifulSoup` 解析 HTML；反爬策略与应对（Headers、延时、重试）
- **实操（30min）**：抓取一个静态网页（如豆瓣图书榜单），提取书名、评分、链接，保存为 CSV
- **交付物**：`day05/book_scraper.py`

### Day 6 — 动态页面 & 批量下载⭐
- **概念（30min）**：静态抓取的局限；`Playwright` 控制浏览器；`asyncio` 异步并发批量下载
- **实操（30min）**：用 Playwright 抓取一个 JavaScript 渲染的页面；批量下载图片/文件到本地
- **交付物**：`day06/dynamic_scraper.py` + `day06/batch_downloader.py`

### Day 7 — 自动化 & 定时调度⭐
- **概念（30min）**：`APScheduler` 定时任务；`Click` 构建专业 CLI；数据存储到 SQLite；项目结构整理
- **实操（30min）**：把 Day 5 的爬虫封装成可定时运行的 CLI 工具，结果存入数据库
- **交付物**：`day07/auto_scraper/`（完整可运行的自动化爬虫工具）

## 第二阶段：TypeScript / Node.js（Day 8-14）

### Day 8 — Node.js 环境 & TypeScript 核心语法
- **概念（30min）**：安装 Node.js（`brew install node`），`npm/pnpm`，TypeScript 编译流程，基本类型、interface、type
- **实操（30min）**：初始化 TS 项目，写脚本处理数组数据（map/filter/reduce）
- **交付物**：`day08/data_transform.ts`

### Day 9 — 函数、泛型、异步
- **概念（30min）**：箭头函数、泛型（`<T>`）、Promise、async/await、fetch API
- **实操（30min）**：用 TS 写一个异步下载脚本，批量获取 URL 列表内容
- **交付物**：`day09/batch_fetch.ts`

### Day 10 — React 核心概念
- **概念（30min）**：JSX、组件、props、useState、useEffect、事件处理
- **实操（30min）**：用 Vite 创建 React+TS 项目，做一个数据展示列表组件
- **交付物**：`day10/data-dashboard/`

### Day 11 — React 数据展示 & 表格
- **概念（30min）**：受控组件、表单过滤、列表渲染、分页逻辑
- **实操（30min）**：制作一个可搜索/过滤的数据表格页面（展示爬取的数据）
- **交付物**：`day11/data-table/`

### Day 12 — 样式 & 布局
- **概念（30min）**：Tailwind CSS 基础、Flexbox、Grid、响应式设计
- **实操（30min）**：给数据表格页面加样式，做到美观可用
- **交付物**：`day12/data-table-styled/`

### Day 13 — 前后端联调
- **概念（30min）**：CORS、fetch 调用后端 API、环境变量、错误处理
- **实操（30min）**：前端调用 Python 后端接口展示数据库中的爬取结果
- **交付物**：`day13/fullstack-scraper-ui/`

### Day 14 — TypeScript 阶段回顾
- **概念（30min）**：回顾 TS 和 React 要点；Puppeteer/Playwright JS 版快速了解
- **实操（30min）**：用 Node.js 写一个简单的网页截图工具（Puppeteer）
- **交付物**：`day14/screenshot-tool/`

## 第三阶段：全栈整合 & AI 辅助（Day 15-21）

### Day 15 — Next.js 入门
- **概念（30min）**：Next.js App Router、文件系统路由、Server Components vs Client Components
- **实操（30min）**：用 `npx create-next-app@latest` 创建项目，搭建数据展示首页
- **交付物**：`day15/scraper-dashboard/`

### Day 16 — API Routes & 数据服务
- **概念（30min）**：Next.js Route Handlers、数据获取模式、Server Actions
- **实操（30min）**：实现接口读取数据库中的爬取结果，供前端展示
- **交付物**：`day16/scraper-dashboard/`（加入 API 层）

### Day 17 — 数据库集成（Prisma）
- **概念（30min）**：Prisma ORM、Schema 定义、Migration
- **实操（30min）**：用 Prisma 管理爬取数据的存储，支持分页查询
- **交付物**：`day17/scraper-dashboard/`（加入数据库）

### Day 18 — AI + 数据处理⭐
- **概念（30min）**：Claude API / Anthropic SDK；用 AI 清洗、提炼、分类爬取到的原始数据
- **实操（30min）**：写一个脚本，把爬取到的网页文本喂给 Claude，提取结构化信息
- **交付物**：`day18/ai_extractor.py`

### Day 19 — 完整自动化平台
- **概念（30min）**：Vercel AI SDK、流式响应；系统集成思路
- **实操（30min）**：整合爬虫 + 数据库 + AI 提取 + Next.js 前端，打通全链路
- **交付物**：`day19/auto-platform/`

### Day 20 — 部署 & 定时触发
- **概念（30min）**：Vercel/Railway 部署；GitHub Actions 定时触发爬虫（cron）；Docker 基础
- **实操（30min）**：部署前端到 Vercel，用 GitHub Actions 每天定时运行爬虫脚本
- **交付物**：`.github/workflows/scraper.yml`

### Day 21 — 个人工具项目启动
- **概念（30min）**：回顾 21 天路线；整理自己会做、不会做的清单
- **实操（30min）**：用 Claude Code 辅助启动一个你实际需要的自动化工具项目
- **交付物**：个人项目初始 repo + CLAUDE.md
