## 1. Python 是什么？

Python 是一门**优雅、简洁、易读**的编程语言，由荷兰人 Guido van Rossum 在 1989 年圣诞节期间发明。

**Python 能做什么？**

- 网站和后台服务（YouTube、Instagram、豆瓣都用 Python）
- 数据分析、人工智能、机器学习
- 自动化脚本、系统管理工具
- 科学计算

**Python 的优缺点**

| 优点 | 缺点 |
|------|------|
| 语法简洁，上手容易 | 运行速度慢（比 C 慢 100 倍以上） |
| 库生态丰富 | 代码不能加密（发布即开源） |
| 跨平台（Windows/Mac/Linux 通用） | GIL 限制多核并行（后续讲解） |

> **Python 的哲学**：用最少的代码做最多的事，写出让人一眼看懂的程序。

## 2. 安装 Python

Python 有 2.x 和 3.x 两个大版本，**请务必安装 3.x**，两者不兼容，本教程全部基于 Python 3。

### 在 Windows 上安装

**方法一（推荐）**：访问 [python.org](https://www.python.org) 下载 `Windows installer (64-bit)`，运行安装包。

> ⚠️ **关键步骤**：安装时务必勾选 **"Add Python 3.x to PATH"**，否则终端找不到 python 命令！

**方法二**：用包管理器 Scoop：

```powershell
scoop install python
```

### 在 macOS 上安装

macOS 系统自带的是 Python 2.x，需要额外安装 3.x：

```bash
# 方法一：用 Homebrew（推荐）
brew install python3

# 方法二：去官网下载 macOS 安装包
```

### 在 Linux 上安装

```bash
# Ubuntu/Debian
sudo apt install python3

# CentOS/RHEL
sudo yum install python3
```

### 验证安装

打开终端（Windows 用 PowerShell，Mac/Linux 用 Terminal）：

```bash
# Windows
python --version

# macOS / Linux
python3 --version
```

看到类似 `Python 3.x.x` 的输出，说明安装成功！

### 进入交互模式

```bash
# Windows
python

# macOS / Linux
python3
```

出现 `>>>` 提示符，就进入了 Python 交互环境，可以直接输入代码并立即看到结果：

```python
>>> 1 + 1
2
>>> print("Hello, Python!")
Hello, Python!
```

输入 `exit()` 并回车，退出交互模式。

## 3. 第一个 Python 程序

### 交互模式 vs 文件模式

| 模式 | 用途 | 提示符 |
|------|------|--------|
| 交互模式 | 临时测试、学习探索 | `>>>` |
| 文件模式 | 写完整程序 | 无 |

### 用文件模式写程序

1. 新建文件 `hello.py`，写入：

```python
print('Hello, World!')
```

2. 在终端运行：

```bash
python3 hello.py
```

输出：
```
Hello, World!
```

### `print()` — 输出

`print()` 是最常用的输出函数：

```python
print('Hello, World!')        # 输出字符串
print(100)                    # 输出整数
print(100 + 200)              # 输出计算结果：300
print('100 + 200 =', 100+200) # 输出：100 + 200 = 300
```

### `input()` — 接收用户输入

```python
name = input('请输入你的名字：')
print('你好，', name)
```

运行后，程序会等待用户输入，回车后继续执行：

```
请输入你的名字：小明
你好，小明
```

> **注意**：`input()` 接收到的永远是**字符串**类型。如果需要数字，要用 `int()` 转换：
> ```python
> age = int(input('请输入年龄：'))
> ```

## 4. Python 基本语法

### 缩进是语法的一部分

Python 用**缩进**（而不是 `{}` 花括号）来表示代码块。约定使用 **4 个空格**缩进：

```python
# 正确写法
if 10 > 5:
    print('10 比 5 大')  # 4 个空格缩进
    print('这行也属于 if 块')

# 错误写法（缩进不一致会报错）
if 10 > 5:
  print('只缩进了 2 个空格')    # 不推荐，可能引发问题
```

### 注释

```python
# 这是单行注释，井号后面的内容不会被执行

"""
这是多行注释
可以写很多行
"""
```

### 大小写敏感

Python 严格区分大小写：

```python
name = 'Tom'
Name = 'Jerry'   # 这是两个不同的变量
print(name)      # 输出 Tom
print(Name)      # 输出 Jerry
```

## 5. 数据类型与变量

### 基本数据类型

#### 整数（int）

```python
a = 100
b = -200
c = 0

# Python 整数没有大小限制
big = 10_000_000_000  # 用下划线分隔，更易读（等于 10000000000）

# 十六进制
hex_num = 0xff00  # 等于十进制 65280
```

**整数运算**：

```python
>>> 10 / 3      # 普通除法，结果是浮点数
3.3333333333333335
>>> 10 // 3     # 地板除（整除），只取整数部分
3
>>> 10 % 3      # 取余数
1
>>> 2 ** 10     # 幂运算（2 的 10 次方）
1024
```

#### 浮点数（float）

```python
pi = 3.14159
e = 2.718

# 科学计数法
big = 1.23e9    # 1.23 × 10^9
small = 1.2e-5  # 0.000012
```

> **注意**：浮点数运算可能有精度误差，如 `0.1 + 0.2 = 0.30000000000000004`，这是正常现象。

#### 字符串（str）

```python
s1 = 'Hello'
s2 = "World"
s3 = '我是中文'
```

（详见第 6 节）

#### 布尔值（bool）

```python
t = True
f = False

# 布尔运算
>>> True and False   # 与：都为 True 才为 True
False
>>> True or False    # 或：有一个 True 就为 True
True
>>> not True         # 非：取反
False

# 比较运算返回布尔值
>>> 3 > 2
True
>>> 5 == 5
True
>>> 5 != 3
True
```

#### 空值（None）

```python
x = None  # 表示"没有值"，不是 0，也不是空字符串
```

### 变量

变量名规则：只能包含字母、数字和 `_`，不能以数字开头，区分大小写。

```python
age = 18           # 整数
name = 'Alice'     # 字符串
is_student = True  # 布尔值
score = 95.5       # 浮点数
```

Python 是**动态语言**，变量类型不固定，可以随时赋值为不同类型：

```python
a = 123     # a 是整数
a = 'ABC'   # a 现在是字符串（合法！）
```

**理解变量的本质**：变量是指向对象的**标签**，不是装数据的盒子：

```python
a = 'ABC'
b = a       # b 和 a 指向同一个 'ABC'
a = 'XYZ'   # a 改变指向，b 不受影响
print(b)    # 输出 'ABC'，不是 'XYZ'
```

### 常量

Python 没有真正的常量机制，约定用**全大写**变量名表示常量：

```python
PI = 3.14159265359
MAX_SIZE = 1000
```

## 6. 字符串详解

### 基本写法

```python
s1 = 'Hello'         # 单引号
s2 = "Hello"         # 双引号（等价）
s3 = "I'm OK"        # 含单引号时用双引号
s4 = 'Say "Hello"'   # 含双引号时用单引号
```

### 转义字符

```python
>>> print('I\'m OK')       # 用 \ 转义单引号
I'm OK
>>> print('Hello\nWorld')  # \n 换行
Hello
World
>>> print('Tab\there')     # \t 制表符
Tab	here
>>> print('C:\\Users')     # \\ 表示一个反斜杠
C:\Users
```

常用转义字符：

| 转义 | 含义 |
|------|------|
| `\n` | 换行 |
| `\t` | 制表符（Tab） |
| `\\` | 反斜杠 `\` |
| `\'` | 单引号 `'` |
| `\"` | 双引号 `"` |

### 原始字符串（r-string）

如果字符串中有大量反斜杠，用 `r''` 前缀可以让内部字符串**不转义**：

```python
>>> print(r'\n\t\\')  # 原样输出
\n\t\\
```

常用于正则表达式和文件路径：

```python
path = r'C:\Users\Tom\Documents'  # Windows 路径
```

### 多行字符串

```python
text = '''第一行
第二行
第三行'''

print(text)
# 输出：
# 第一行
# 第二行
# 第三行
```

### 字符编码

Python 3 的字符串使用 **Unicode** 编码，天然支持中文、日文等所有语言。

```python
>>> len('ABC')    # 英文字符，每个长度为 1
3
>>> len('中文')   # 中文字符，也是 1
2
```

**bytes 和 str 的互转**：

```python
# str → bytes（编码）
>>> 'ABC'.encode('utf-8')
b'ABC'
>>> '中文'.encode('utf-8')
b'\xe4\xb8\xad\xe6\x96\x87'

# bytes → str（解码）
>>> b'\xe4\xb8\xad\xe6\x96\x87'.decode('utf-8')
'中文'
```

> **记住**：处理中文时，始终使用 **UTF-8** 编码。在 `.py` 文件开头加上：
> ```python
> # -*- coding: utf-8 -*-
> ```

### 字符串格式化

**方法一：% 格式化（旧式）**

```python
name = '小明'
score = 95
print('你好，%s，你的成绩是 %d 分' % (name, score))
# 输出：你好，小明，你的成绩是 95 分
```

常用占位符：`%s`（字符串）、`%d`（整数）、`%f`（浮点数）

**方法二：f-string（推荐，Python 3.6+）**

```python
name = '小明'
score = 95.5
print(f'你好，{name}，你的成绩是 {score:.1f} 分')
# 输出：你好，小明，你的成绩是 95.5 分

r = 2.5
area = 3.14 * r ** 2
print(f'半径为 {r} 的圆面积是 {area:.2f}')
# 输出：半径为 2.5 的圆面积是 19.62
```

f-string 中 `{变量:.2f}` 表示保留 2 位小数，`{变量:>10}` 表示右对齐占 10 个字符宽度。

## 7. 列表（list）和元组（tuple）

### list — 有序可变集合

list 是 Python 最常用的数据类型，**有序、可重复、可修改**。

**创建 list**：

```python
names = ['Michael', 'Bob', 'Tracy']
numbers = [1, 2, 3, 4, 5]
mixed = ['Alice', 18, True, 3.14]  # 元素类型可以不同
empty = []                          # 空 list
```

**访问元素**（索引从 0 开始）：

```python
>>> names = ['Michael', 'Bob', 'Tracy']
>>> names[0]      # 第一个元素
'Michael'
>>> names[-1]     # 最后一个元素（倒数第一）
'Tracy'
>>> names[-2]     # 倒数第二
'Bob'
>>> len(names)    # 获取长度
3
```

**修改、添加、删除**：

```python
names = ['Michael', 'Bob', 'Tracy']

# 修改
names[1] = 'Sarah'
print(names)  # ['Michael', 'Sarah', 'Tracy']

# 末尾追加
names.append('Adam')
print(names)  # ['Michael', 'Sarah', 'Tracy', 'Adam']

# 指定位置插入
names.insert(1, 'Jack')
print(names)  # ['Michael', 'Jack', 'Sarah', 'Tracy', 'Adam']

# 删除末尾元素（并返回该元素）
last = names.pop()
print(last)   # 'Adam'

# 删除指定位置元素
names.pop(1)  # 删除索引 1 的 'Jack'
print(names)  # ['Michael', 'Sarah', 'Tracy']
```

**嵌套 list**（二维数组）：

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print(matrix[1][2])  # 输出 6（第二行第三列）
```

### tuple — 有序不可变集合

tuple 和 list 类似，但**一旦创建，不能修改**（没有 append、pop 等方法）。

```python
>>> t = (1, 2, 3)
>>> t[0]
1
>>> t[0] = 99     # 报错！tuple 不可修改
TypeError: 'tuple' object does not support item assignment
```

**为什么用 tuple？** 因为 tuple 不可变，更安全，适合存储不应被修改的数据。

**创建 tuple 的坑**：

```python
t1 = ()       # 空 tuple
t2 = (1,)     # 只有一个元素，必须加逗号！
t3 = (1)      # 这不是 tuple，是整数 1！
```

> 记住：单元素 tuple 必须写 `(元素,)`，加逗号。

## 8. 条件判断与模式匹配

### if / elif / else

```python
age = 18

if age >= 18:
    print('成年人')
elif age >= 12:
    print('青少年')
else:
    print('儿童')
```

**完整语法**：

```python
if 条件1:
    代码块1
elif 条件2:
    代码块2
elif 条件3:
    代码块3
else:
    代码块4  # 以上条件都不满足时执行
```

**简写条件**：

```python
x = 5
if x:            # 非零、非空字符串、非空列表，都被当作 True
    print('x 不为空/零')
```

**常见坑 — input 返回字符串**：

```python
# 错误写法
birth = input('请输入出生年份：')
if birth < 2000:    # 报错！str 不能和 int 比较
    print('00 前')

# 正确写法
birth = int(input('请输入出生年份：'))
if birth < 2000:
    print('00 前')
else:
    print('00 后')
```

### match 语句（Python 3.10+）

当需要对一个变量做多种情况匹配时，用 `match` 比 `if-elif` 更清晰：

```python
score = 'B'
match score:
    case 'A':
        print('优秀')
    case 'B':
        print('良好')
    case 'C':
        print('及格')
    case _:   # _ 表示"其他所有情况"
        print('无效成绩')
```

**复杂匹配**：

```python
age = 15
match age:
    case x if x < 10:           # 带条件的 case
        print(f'{x} 岁以下')
    case 10:
        print('10 岁')
    case 11 | 12 | 13 | 14 | 15 | 16 | 17 | 18:  # 多值匹配
        print('11~18 岁')
    case _:
        print('18 岁以上')
```

## 9. 循环

### for 循环

`for...in` 循环依次取出可迭代对象的每个元素：

```python
names = ['Michael', 'Bob', 'Tracy']
for name in names:
    print('Hello,', name)
# 输出：
# Hello, Michael
# Hello, Bob
# Hello, Tracy
```

**`range()` 函数**：生成整数序列

```python
# range(n) 生成 0 到 n-1
for i in range(5):
    print(i)
# 输出 0 1 2 3 4

# 计算 1+2+...+100
total = 0
for x in range(1, 101):  # range(1, 101) 生成 1 到 100
    total += x
print(total)  # 5050
```

### while 循环

只要条件为 True，就一直循环：

```python
n = 1
while n <= 10:
    print(n)
    n += 1  # 等同于 n = n + 1
```

### break 和 continue

```python
# break — 立即退出整个循环
n = 1
while n <= 100:
    if n > 10:
        break   # 当 n > 10 时，退出循环
    print(n)
    n += 1
# 只输出 1 到 10

# continue — 跳过本次循环，继续下一次
for n in range(1, 11):
    if n % 2 == 0:
        continue  # 偶数时跳过
    print(n)
# 只输出奇数：1 3 5 7 9
```

> **建议**：不要滥用 break 和 continue，会让代码逻辑难以理解。大多数情况可以通过修改循环条件来避免。

## 10. 字典（dict）和集合（set）

### dict — 键值对存储

dict（字典）使用**键（key）—值（value）**的方式存储数据，查找速度极快（原理是哈希算法）：

```python
# 创建 dict
scores = {'Michael': 95, 'Bob': 75, 'Tracy': 85}

# 访问
>>> scores['Michael']
95

# 添加/修改
scores['Adam'] = 67
scores['Michael'] = 99  # 修改已有 key 的值

# 判断 key 是否存在
>>> 'Thomas' in scores
False

# 安全获取（key 不存在时返回默认值，不报错）
>>> scores.get('Thomas', 0)
0

# 删除
scores.pop('Bob')

# 遍历
for name, score in scores.items():
    print(f'{name}: {score}')
```

**dict vs list 对比**：

| | dict | list |
|--|------|------|
| 查找速度 | 极快（O(1)） | 随数据量增大变慢（O(n)） |
| 内存占用 | 较大 | 较小 |
| 适合场景 | 需要快速查找 | 有序数据 |

**重要**：dict 的 key 必须是**不可变对象**（字符串、整数、tuple 等），不能用 list 作为 key。

### set — 无序不重复集合

set 和 dict 类似，但只存 key，不存 value。自动去重：

```python
# 创建 set
s = {1, 2, 3}
s2 = set([1, 1, 2, 2, 3, 3])  # 从 list 创建，自动去重
print(s2)  # {1, 2, 3}

# 添加
s.add(4)

# 删除
s.remove(4)

# 集合运算
s1 = {1, 2, 3}
s2 = {2, 3, 4}
print(s1 & s2)  # 交集：{2, 3}
print(s1 | s2)  # 并集：{1, 2, 3, 4}
print(s1 - s2)  # 差集：{1}
```
