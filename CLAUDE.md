# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

Jerry's Knowledge Base 是个人知识库，用于积累学习文档并托管到 GitHub。所有文档使用中文编写。

## 目录结构

```
├── ai-coding/       # AI 辅助编程（Claude Code、Copilot、Cursor 等）
├── programming/     # 编程语言和技术
├── tools/           # 开发工具使用指南
├── notes/           # 学习笔记
└── resources/       # 资源收集与整理
```

## 文档约定

- 文档使用 Markdown 格式，中文编写
- 每个子目录包含 README.md 作为目录索引
- 文档末尾标注更新日期：`> 📅 最后更新：YYYY-MM-DD`
- 新增内容时更新对应目录的 README.md 操作日志

## 常用命令

```bash
# 查看状态
git status

# 提交更改
git add . && git commit -m "描述"

# 推送到 GitHub
git push origin master
```
