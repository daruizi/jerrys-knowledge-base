# 第 04 章：三层记忆系统

## 概述

Hermes Agent 拥有业界少有的**三层记忆系统**，解决了传统 AI 助手"每次对话从零开始"的根本问题。

| 层级 | 类比 | 存储内容 | 生命周期 |
|------|------|---------|---------|
| **短期记忆** | 工作记忆 | 当前会话上下文 | 单次会话 |
| **程序性记忆** | 技能 | Skill Documents（从经验中提炼） | 持久，可改进 |
| **长期记忆** | 个人档案 | 用户偏好、项目信息、历史模式 | 持久，定期"提示"维持 |

---

## 第一层：短期记忆（Session Context）

- 当前活跃会话中的对话历史
- Agent 在此会话中执行的工记录
- 会话结束时保存到 SQLite 数据库
- 通过 `hermes --continue` 可恢复

**特点：** 容量受模型上下文窗口限制（至少 64K token），超出后自动压缩。

---

## 第二层：程序性记忆（Skills）

技能是 Hermes Agent 从**实际经验**中提炼的可复用能力：

### 技能文档（Skill Document）

完成复杂任务后，Agent 会：

1. 分析任务执行过程
2. 提取关键步骤和决策
3. 编写结构化技能文档
4. 保存到技能目录

### 技能的自我改进

技能不是静态的。Agent 在使用过程中会：

- 发现技能描述不够清晰 → 改进
- 发现步骤有遗漏 → 补充
- 发现更高效的方法 → 更新

### 技能示例

```markdown
# 代码审查技能

## 触发条件
当用户请求代码审查时

## 执行步骤
1. 读取目标文件
2. 检查代码风格
3. 识别潜在 bug
4. 评估性能影响
5. 给出改进建议

## 注意事项
- 优先关注安全问题
- 区分风格偏好和真正的问题
```

### 技能管理命令

```bash
# CLI 内
/skills                    # 列出所有技能
/skills search kubernetes  # 搜索技能
/skills install <name>     # 安装技能

# 终端命令
hermes skills search kubernetes
hermes skills install openai/skills/k8s
```

## 第三层：长期记忆（Long-term Memory）

### 记忆文件

```
~/.hermes/memory/
├── MEMORY.md    # Agent 自动维护的记忆
└── USER.md      # 用户档案
```

### 记忆类型

| 类型 | 示例 |
|------|------|
| 用户偏好 | "喜欢简洁的代码风格"、"偏好 Python" |
| 项目信息 | "项目使用 FastAPI 框架" |
| 历史模式 | "之前解决过类似的性能问题" |
| 决策记录 | "选择了方案 A 因为..." |

### "提示"机制（Nudges）

长期记忆会随时间淡化。Hermes Agent 定期对自己进行"记忆提示"，将重要信息重新激活，防止遗忘关键上下文。

### Honcho 辩证用户建模

Hermes Agent 集成了 [Honcho](https://github.com/plastic-labs/honcho)：

- **辩证方法**：通过对话中的正反两面分析建立用户模型
- **深度理解**：不只是记录事实，而是理解用户的思维模式
- **跨会话学习**：每次对话都丰富用户画像

---

## 记忆系统工作流

```
用户对话 → 短期记忆记录
                ↓
完成复杂任务 → 提炼为技能（程序性记忆）
                ↓
发现用户偏好 → 写入长期记忆
                ↓
下次对话 → 检索相关记忆
                ↓
新对话中应用记忆 → 更个性化的响应
```

---

## 记忆 Provider 插件

Hermes Agent 支持自定义记忆后端：

```yaml
# config.yaml 配置
memory:
  provider: sqlite        # 默认：SQLite
  # 或自定义 provider
```

支持的记忆 Provider：
- **SQLite**（默认）：本地文件存储
- **Honcho**：辩证用户建模
- **自定义插件**：通过 `~/.hermes/plugins/memory/` 加载

---

## 记忆 vs 技能的区别

| 维度 | 记忆（Memory） | 技能（Skills） |
|------|---------------|---------------|
| 内容 | "你是谁" | "怎么做" |
| 格式 | 自由文本 | 结构化文档 |
| 用途 | 个性化响应 | 可复用工作流 |
| 触发 | 自动检索 | 按需加载 |
| 来源 | 对话中自动提取 | 从经验中主动提炼 |
