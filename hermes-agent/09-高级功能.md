# 第 09 章：高级功能

## Cron 定时任务

### 概述

Hermes Agent 内置 Cron 调度器，可以按时间表自动执行任务。Cron 任务是**第一类 Agent 任务**（不是 shell 脚本），可以附加技能、投递到任意平台。

### 基本用法

```bash
# 在对话中设置
"/cron 每天早上 8 点，给我发送今日天气摘要"

# 或通过命令
/cron set "0 8 * * *" "给我发送今日新闻摘要"
/cron list           # 列出所有定时任务
/cron delete <id>    # 删除定时任务
/cron run <id>       # 立即执行一次
```

### Cron 调度格式

```
┌─── 分钟 (0-59)
│ ┌─── 小时 (0-23)
│ │ ┌─── 日期 (1-31)
│ │ │ ┌─── 月份 (1-12)
│ │ │ │ ┌─── 星期 (0-7, 0和7=周日)
│ │ │ │ │
* * * * *  任务描述
```

### 常见示例

```bash
# 每天早上 9 点
0 9 * * *

# 每 30 分钟
*/30 * * * *

# 工作日早上 8 点
0 8 * * 1-5

# 每月 1 号
0 0 1 * *
```

## 子 Agent 委派

### 概述

Hermes Agent 可以**并行派生隔离子 Agent**，每个子 Agent 独立执行不同工作流。

### 使用场景

| 场景 | 说明 |
|------|------|
| 并行调研 | "调研 A、B、C 三个方案，分别出报告" |
| 代码分析 | 一个 Agent 读源码，一个写测试，一个文档 |
| 多任务处理 | 同时处理多个用户需求 |

### 示例

```
帮我并行分析这个项目的三个模块：
1. 认证模块的安全性
2. API 模块的性能
3. 数据库模块的扩展性

派三个子 Agent 分别分析，最后汇总。
```

## 浏览器自动化

### 概述

Hermes Agent 支持完整的浏览器自动化，包括：

- 网页浏览和截图
- 点击、填写表单
- 抓取网页内容
- 视觉理解（截图分析）

### 浏览器后端

| 后端 | 说明 |
|------|------|
| playwright | 推荐，功能最全 |
| puppeteer | Chrome/Chromium |
| 其他 | 根据需求选择 |

### 浏览器工具

```
browser_navigate  https://example.com   # 导航
browser_click      "登录按钮"             # 点击
browser_fill       "用户名" "admin"      # 填写
browser_screenshot                       # 截图
browser_extract                            # 提取内容
```

## RL 训练（Atropos）

### 概述

Nous Research 的 Atropos 强化学习框架可以对 Hermes Agent 进行微调，优化其行为。

### 适用场景

- 优化 Agent 在特定领域（如代码审查、安全审计）的表现
- 批量生成训练轨迹
- 实验不同的 Agent 策略

### 基本流程

```bash
# 1. 生成训练数据
hermes rl generate --output trajectories.json

# 2. 运行 RL 训练
hermes rl train --data trajectories.json --model base-model

# 3. 评估
hermes rl eval --model fine-tuned-model
```

