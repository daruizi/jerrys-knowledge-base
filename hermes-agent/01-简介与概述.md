# 第 01 章：Hermes Agent 简介与概述

## 什么是 Hermes Agent？

Hermes Agent 是由 **Nous Research** 开发的开源自进化 AI Agent，2026 年 2 月 25 日首次发布。它的定位不是 IDE 里的代码助手，也不是套壳聊天机器人，而是一个**完全自主的 Agent**——随着使用时间的增长而变得越来越强大。

核心理念：**The agent that grows with you.**

## 核心特性

### 1. 闭环学习系统（核心差异化能力）

这是 Hermes Agent 与其他 Agent 最大的区别。它拥有一个**内置的学习闭环**：

- **Agent 管理的记忆**：自动记录重要信息，定期"提示"自己保持关键知识
- **自主技能创建**：完成复杂任务后，自动将经验提炼为可复用的"技能文档"
- **技能自我改进**：使用过程中不断优化已有技能
- **跨会话回忆**：基于 FTS5 全文搜索的跨会话记忆检索 + LLM 摘要
- **Honcho 辩证用户建模**：深度理解用户偏好和工作模式

## 为什么需要 Hermes Agent？

**传统 AI 助手的痛点：**

1. **健忘**：每次对话从零开始，不记得之前的上下文
2. **不成长**：用了一年和第一天一样的水平
3. **范围受限**：只能在 IDE 或聊天窗口里工作
4. **模型锁定**：绑定到单一 Provider
5. **不自主**：只能被动响应，不能主动执行

**Hermes Agent 的解决方案：**

1. ✅ **三层记忆**：短期、程序性、长期记忆，越用越了解你
2. ✅ **学习闭环**：从经验中提炼技能，用中改进
3. ✅ **无处不在**：跑在 VPS、云端、手机（Termux）
4. ✅ **模型无关**：随时切换 Provider，无锁定
5. ✅ **自主执行**：Cron 定时任务、主动工具调用、子 Agent 委派
