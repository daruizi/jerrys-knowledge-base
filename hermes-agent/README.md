# Hermes Agent 学习指南

> **Hermes Agent** — 由 [Nous Research](https://nousresearch.com) 打造的自进化 AI Agent。
> 全球唯一内置**学习闭环**的 Agent：从经验中创建技能、在使用中改进技能、跨会话保持记忆。

## 学习路径推荐

| 级别 | 目标 | 推荐阅读顺序 | 预计时间 |
|------|------|-------------|---------|
| **入门** | 安装运行、基本对话、内置工具 | 01 → 02 → 06（工具概览） | ~1 小时 |
| **进阶** | 消息机器人、记忆、Cron、技能 | 02 → 04 → 05 → 07 → 09 | ~2-3 小时 |
| **高级** | 自定义工具/技能、RL 训练、二次开发 | 03 → 05 → 06 → 10 | ~4-6 小时 |
